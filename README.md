# Kelompok 8 - Mikrokontroler

## Tema Project: Alat Siram Otomatis Menggunakan Soil Sensor

Dirancang untuk membantu proses penyiraman tanaman secara otomatis berdasarkan tingkat kelembapan tanah. Sistem ini menggunakan **soil moisture sensor** untuk mendeteksi kondisi tanah, sehingga tanaman dapat memperoleh air sesuai kebutuhan tanpa harus disiram secara manual.

MQTT digunakan sebagai protokol komunikasi data. Melalui MQTT, data kelembapan tanah, status pompa, dan kondisi penyiraman dapat dikirim secara real-time ke aplikasi monitoring, seperti dashboard IoT atau smartphone. Pengguna dapat memantau kondisi tanaman dari jarak jauh melalui jaringan internet.

---

## Komponen Utama

- ESP32
- Soil Sensor
- Relay 1 Module 5 Volt
- Power Supply (Baterai)
- Laptop (Monitoring)
- Konektor Baterai
- Kabel (Male dan Female)
- Pompa Air 5 Volt

## Komponen Tambahan

- Obeng
- Ampere Meter (Mengecek tegangan voltase pin 5V ESP32)
